python crawl

URL 管理器
实现方式：内存、关系数据库、缓存数据库、redis
内存：将爬取的URL存储在python内存中，即存储在set()集合中。
关系数据库：将URL存储在urls(url, is_crawled)。
redis将已爬取和未爬取的URL分别存储在redis的set()集合中。

网页下载器
网页下载器会将需要爬取的网页已html的方式将网页下载存储至本地文件或内存字符串。
下载器：urllib2(官方基础模块) 和 requests(第三方库)

urllib2网页下载的方法：
1、直接打开网页：

response = urllib2.urlopen(url)
response.getcode() # 获取状态码，当200时表示成功
content = response.read() # 读取网页的内容

2、添加data、header:

# 创建request对象
request = urllib2.Request(url)
#添加数据：
request.add_data("a", 1)
# 添加请求头
request.add_header('User-Agent', 'Mozilla/5.0')
# 发送请求获取结果
content = urllib2.urlopen(request)

3、添加特殊情景的处理器:

# 创建cookie容器
cj = cookielib.CookieJar()
# 创建opener
opener = urllib2.build_opener(urllib2.HTTPCookieProcessor(cj))
# 给urllib2安装opener
urllib2.install_opener(opener)
# 使用带有cookie的urllib2访问网页
response = urllib2.urlopen(url)

网页解析器
网页解析器就是以爬取的html字符串为输入，分析出有价值的数据和URL。
解析器的分类有：正则表达式、html.parser、Beautiful Soul、lxml

beautiful Soul解析器，将html传给BeautifulSoul(html),创建一个BeautifulSoul对象，以便于操作节点。

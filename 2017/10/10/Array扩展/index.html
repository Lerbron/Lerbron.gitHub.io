Array扩展

数组新增的方法：

1) find()用于找出第一个符合条件的数组成员,返回该成员。该方法接受一个回调函数，回调函数可以有三个形参,依次为当前的值、当前的位置和原数组

let a = [1, 5, 10, 15].find(function(value, index, arr) {
  return value > 9;
});
console.log(a);   // 10

2) findIndex(),用于找出第一个符合条件的数组成员,返回该成员的位置。该方法接受一个回调函数，回调函数可以有三个形参,依次为当前的值、当前的位置和原数组

let b = [1, 5, 10, 15].findIndex(function(value, index, arr) {
  return value > 9;
});
console.log(b);  // 2

3) includes() 用于检测数组是否包含某个值，返回boolean。

let c = [1, 2, 5, 7].includes(2);
console.log(c);   // true

4) Array.of() 将一组值转换为数组

let arrOf = Array.of(2, 4, 5);
console.log(arrOf);   // [2, 4, 5]

5) Array.from() 将类数组和可遍历数组转换为真正的数组

6) Array.entries() 对数组的键值对进行遍历

let arr = [2, 4, 6];
for (let [idx, val] of arr.entries()) {
      console.log(idx, val);
      // 0 2
    // 1 4
    // 2 6
}

7) Array.keys() 对数组的键进行遍历

for (let idx of arr.keys()) {
    console.log(idx); // 0  1   2
}

8) Array.values() 对数组的值进行遍历

for (let value of arr.values()) {
      console.log(value);   // 2  4  6
}

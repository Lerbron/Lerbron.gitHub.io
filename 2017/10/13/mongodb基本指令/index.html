mongodb基本指令

1、数据库连接：sudo mongod
2、查询显示所有数据库：show dbs
3、查询显示所有集合：show collections
4、创建／切换数据库：use
5、删除当前数据库：db.dropDatabase().需要切换至当前要删除的数据库
6、查询数据库指定集合中的所有数据：db.collection.find()
7、按条件查询数据库指定集合中的数据：

db.collection.find({'name': 'admin'})    查询name为admin的数据。

8、创建集合：

db.createCollection('collectionName')

9、删除集合：

db.collectionName.drop()

10、查询当前集合的数据条数：

db.collectionName.count()

11、查询当前集合中的第一条数据：

db.collectionName.findOne()

12、添加数据列：

db.collectionName.save()

13、修改数据：

db.collectionName.update({'age': 25}, {$set: {'name': 'Lee'}})  更改age为25的数据的name为Lee

14、更新不存在时，插入数据：

db.collectionName.update({'age': 30}, {$set: {'name': 'bruce'}}, true)

15、更行多条数据：

db.collectionName.update({name: 'one'}, {$set: {name: 'two'}}, false, true)

16、删除数据：

db.collectionName.remove({name: 'two'})

17、查询集合的索引：

db.collectionName.getIndex()

18、创建索引：

db.collectionName.ensureIndex({key: "text"})
db.users.ensureIndex({'name': 'text'})  // 在users的集合上创建键为name的索引

19、获取集合的索引：db.collectionName.getIndexes()

db.users.getIndexes()

[
    {
        "v" : 2,
        "key" : {
            "_id" : 1
        },
        "name" : "_id_",
        "ns" : "test.users"
    }
]

20、全局索引查询：db.collectionName.find({$text: {$search:’aa’}})

db.users.find({$text: {$search:'aa'}})  // 在users的集合中查询含有'aa'的数据
db.users.find({$text: {$search: 'aa bb cc'}})   // 在users的集合中查询含有'aa'或'bb'或'cc'的数据
db.users.find({$text: {$search: 'aa bb -cc'}})  // 在users中查询包含'aa'或'bb'但不包含'cc'的数据
db.users.find({$text: {$search:'\'aa\' \'bb\''}})   // 在users中查询既包含aa也包含'cc'的数据

21、全文索引相似度查询：

db.users.find({$text:{$search: 'aa'}}, {score: {$meta:'textScore'}})    // 查询包含'aa'的数据，数据包含相似度的值
db.users.find({$text: {$search: 'aa'}},{score:{$meta: 'textScore'}}).sort({score:{$meta: 'textScore'}})     // 查询包含'aa'的数据,并对相似度进行排序，相似度越高，越靠前

全文索引的限制：
1）每次查询只能指定一个$text查询
2）$text查询不能出现在$nor查询中
3）查询中如果包含了$text，则hint（指定索引）不起作用
4）全文索引不支持中文查询

23、删除索引：db.collectionName.dropIndex(‘name’) 删除索引名为name的索引

24、索引的属性：

1）指定索引名： db.collectionName.ensureIndex({'x': 1}, {'name': 'mm'})  // 指定索引的名为 mm
2) 唯一性：    db.collectionName.ensureIndex({'x': 1}, {unique: true})  // 不允许插入键为x并值相同的数据
3）稀疏性：    db.collectionName.ensureIndex({'x': 1}, {'sparse':true}) // 当插入的数据不含有x字段时，不为该条数据创建索引。默认 sparse：false。

25、查询值存在某个字段的数据：

db.colleciontName.find({name: {$exists: true}})  // 查询存在name字段的数据

26、地理位置索引：
2d平面位置索引：

db.collectionName.ensureIndex({w: '2d'})
db.collectionName.insert({w: [100, 80]})    // 插入经度为100，纬度为80的数据
db.collectionName.find({w: {$near: [100, 80]}})     // 查询经度为100，纬度为80附近的点，最多返回100个改点附近的点
db.collectionName.find({w: {$near: [100, 80], $maxDistance: 10}})   // 查询经度为100，纬度为80附近的点,但最大距离为10的点
db.collectionName.find({w: {$geoWithin: {$box: [[x1,y1],[x2, y2]]}}})   // 查询左边界为x1,y1、右边界为x2,y2范围内的点
db.collectionName.find({w: {$geoWithin: {$center: [[x1,y1],r]}}})       // 查询圆心为x1,y1，半径为r范围内的点
db.collectionName.find({w: {$geoWithin: {$polygon: [[x1,y1],[x2, y2],[x3,y3]]}}})   // 查询点（x1, y1）(x2,y2) (x3, y3)组成的多边形内的点
db.runCommand({geoNear:collectionName, near:[x,y], minDistance: num, MaxDistance: num, num: num})   // minDistance对2d无效

2dsphere位置索引

db.collectionName.ensureIndex({loc: '2dsphere'})
db.collectionName.find( { loc :  { $near: { $geometry : { type : "Point", coordinates : [ <longitude> , <latitude> ] },$maxDistance : <distance in meters>} } } )

27、导入json文件：

mongoimport -d test -c list --file /Users/admin/Desktop/list.json   往test数据库中的list集合导入json文件。该文件为json对象
mongoimport -d test -c list --file /Users/admin/Desktop/list.json --jsonArray   往test数据库中的list集合导入json文件。该文件为json数组

redux理解

一、combineReducers.js
combineReducers接受一个参数，改参数为对象，返回一个函数combination

combination函数接收当前state树以及一个action对象作为参数。该方法主要是遍历所有的reducer方法，并将当前的state树和action传入reducer方法中，返回新的state树。当新的state树有变化时，更新并返回新的state树。

二、applyMiddleware.js
applyMiddleware函数接收若干个中间件的函数作为参数，最终返回一个增强的dispatch和store。

applyMiddleware方法的使用：

const store = createStore(
  reducer,
  preloadedState,
  applyMiddleware(...middleware)
)

// createStore 方法
export default function createStore(reducer, preloadedState, enhancer) {
  if (typeof preloadedState === 'function' && typeof enhancer === 'undefined') {
    enhancer = preloadedState
    preloadedState = undefined
  }

  if (typeof enhancer !== 'undefined') {
    if (typeof enhancer !== 'function') {
      throw new Error('Expected the enhancer to be a function.')
    }

    /* 将createStore传入到applyMiddleware(...middleware)返回的匿名函数中，同时传入参数reducer再调用一次.             preloadedState
    */
    return enhancer(createStore)(reducer, preloadedState)
  }

  ...

}

三、createStore.js
createStore方法接收三个参数，reducers/initSate/enhancer，方法中定义了getState、subscribe、dispatch、replaceReducer、observable方法，并返回返回一个包含这些方法的对象。

getState：返回当前的state树
subscribe：将所有的订阅了store中的数据的listener添加到nextListeners中，并返回一个可取消订阅的句柄函数。
replaceReducer：替换计算state的reducer

express

express 是一个路由和中间件web框架，express应用程序基本上都是一系列中间件函数的调用。
中间件：其实就是一个函数，在响应请求之前，对请求进行的一些操作。该函数能够访问请求对象、响应对象以及下一个中间件函数。因此在中间件函数中可操做的事物：
1）执行任何代码；
2）更改请求对象和响应对象；
3）结束请求/响应循环；
4）调用堆栈中的下一个中间件函数。

中间件的类型：
1）应用层中间件；应用层中间件是使用app.use()或app.METHOD()函数，将应用层中间件绑定到应用程序对象的实例，其中 METHOD 是中间件函数处理的请求的小写 HTTP 方法（例如 GET、PUT 或 POST）。
2）路由中间件；路由层中间件和应用层中间件基本相同，差异之处它是绑定到express.Router() 的实例。使用 router.use() 和 router.METHOD() 函数装入路由器层中间件。
3）错误处理中间件；错误处理中间件函数的定义方式与其他中间件函数基本相同，差别在于错误处理函数有四个自变量而不是三个，专门具有特征符 (err, req, res, next)
4）内置中间件；
5）第三方中间件

TREEJS

一、相机主要分为正交投影相机和透视投影相机。

正交投影相机：
THREE.OrthographicCamera(left, right, top, bottom, near, far),为了保证照相机的横竖比，需保证（right - left）与 (top - bottom)的比例与canvas宽高比一致。

设置照相机：
    let camera = new THREE.OrthographicCamera(-5, 5, 5, -5, 1, 10);  // 设置视景体
    camera.positio.set(0, 0, 10);   // 设置相机位置
    sceme.add(camera);  // 将相机添加到场景中

透视投影相机：
THREE.PerspectiveCamera(fov, aspect, near, far),fov是视景体竖直方向的张角，aspect等于width/height，是相机水平方向和竖直长度的比值，near和far分别是照相机到视景体最近、最远的距离，均为正值

// test

设置透视相机：
    let camera = new THREE.PerspectiveCamera(45, 400 / 300, 1, 10);
    camera.position.set(0, 0, 5);
    scene.add(camera);

二、几何形状
几何形状最主要的功能就是储存一个物体的顶点信息。

立方体：
    THREE.CubeGeometry(width, height, depth, widthSegments, heightSegments, depthSegments)
        width: x方向上的长度
        height: y方向上的长度
        depth: Z方向上的长度
        后三个参数分别是在三个方向上的分段数

平面：其实是一个长方形，而不是数学上的无限大小的平面
   THREE.PlaneGeometry(width, height, widthSegments, heightSegments)
        width: x方向上的长度
        height: y方向上的长度
        后面的两个参数表示在两个方向上的分段数

球体：
   THREE.SphereGeometry(radius, segmentsWidth, segmentsHeight, phiStart, phiLength, thetaStart, thetaLength)
       radius是半径；
       segmentsWidth：经度上的切片数；
       segmentsHeight：纬度上的切片数；
       phiStart：经度开始的弧度；
       phiLength：经度跨过的弧度；
       thetaStart：纬度开始的弧度；
       thetaLength：纬度跨过的弧度

圆形：
   THREE.CircleGeometry(radius, segments, thetaStart, thetaLength)

圆柱体：
    THREE.CylinderGeometry(radiusTop, radiusBottom, height, radiusSegments, heightSegments, openEnded)
         radiusTop与radiusBottom：分别是顶面和底面的半径；height：圆柱体的高度；
         radiusSegments与heightSegments：可类比球体中的分段；openEnded是一个布尔值，表示是否没有顶面和底面，缺省值为false，表示有顶面和底面。

正四面体、正八面体、正十二面体：
    THREE.TetrahedronGeometry(radius, detail)
    THREE.OctahedronGeometry(radius, detail)
    THREE.IcosahedronGeometry(radius, detail)

圆环面：


THREE.TorusGeometry(radius, tube, radialSegments, tubularSegments, arc)
radius：圆环半径；
tube：管道半径；
radialSegments与tubularSegments分别是两个分段数；
arc：圆环面的弧度，缺省值为Math.PI * 2

三、材质
材质是独立于物体顶点信息之外的与与渲染效果相关的属性。通过设置材质可以改变物体的颜色、纹理、光照模式。

基本材质（BasicMaterial）的物体，渲染后物体的颜色始终为该材质的颜色，而不会由于光照产生明暗、阴影效果。
THREE.MeshBasicMaterial(opt)
opt常见属性：
visible：是否可见，默认为true
side：渲染面片正面或是反面，默认为正面THREE.FrontSide，可设置为反面THREE.BackSide，或双面THREE.DoubleSide
wireframe：是否渲染线而非面，默认为false
color：十六进制RGB颜色，如红色表示为0xff0000
map：使用纹理贴图

Lambert材质（MeshLambertMaterial）是符合Lambert光照模型的材质。Lambert光照模型的主要特点是只考虑漫反射而不考虑镜面反射的效果，因而对于金属、镜子等需要镜面反射效果的物体就不适应，对于其他大部分物体的漫反射效果都是适用的。

Phong材质（MeshPhongMaterial）是符合Phong光照模型的材质。和Lambert不同的是，Phong模型考虑了镜面反射的效果，因此对于金属、镜面的表现尤为适合。

法向材质可以将材质的颜色设置为其法向量的方向。

材质贴图：
let texture = THREE.ImageUtils.loadTexture(‘../img/0.png’);
let material = new THREE.MeshLambertMaterial({map:texture});

分别导入图像到六个纹理：
    var materials = [];

for (var i = 0; i < 6; ++i) {
materials.push(new THREE.MeshBasicMaterial({map:THREE.ImageUtils.loadTexture(‘../img/‘ + i + ‘.png’{},function() {renderer.render(scene, camera);}),overdraw: true}));}

var cube = new THREE.Mesh(new THREE.CubeGeometry(5, 5, 5),
new THREE.MeshFaceMaterial(materials));
scene.add(cube);

五、网格
网格是由顶点、边、面等组成的物体；其他物体包括线段、骨骼、粒子系统等。创建物体需要指定几何形状和材质。
创建网格： Mesh(geometry, material)

六、动画
动画的实现通过在美妙中多次重绘画面。当每秒帧数 fps小于20时，一般能感受到画面的卡滞现象。

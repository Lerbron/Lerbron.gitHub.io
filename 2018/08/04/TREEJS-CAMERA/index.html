TREEJS-CAMERA

相机主要分为正交投影相机和透视投影相机。

正交投影相机：
THREE.OrthographicCamera(left, right, top, bottom, near, far),为了保证照相机的横竖比，需保证（right - left）与 (top - bottom)的比例与canvas宽高比一致。

设置照相机：
    let camera = new THREE.OrthographicCamera(-5, 5, 5, -5, 1, 10);  // 设置视景体
    camera.positio.set(0, 0, 10);   // 设置相机位置
    sceme.add(camera);  // 将相机添加到场景中

透视投影相机：
THREE.PerspectiveCamera(fov, aspect, near, far),fov是视景体竖直方向的张角，aspect等于width/height，是相机水平方向和竖直长度的比值，near和far分别是照相机到视景体最近、最远的距离，均为正值

设置透视相机：
    let camera = new THREE.PerspectiveCamera(45, 400 / 300, 1, 10);
    camera.position.set(0, 0, 5);
    scene.add(camera);

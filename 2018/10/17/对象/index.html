对象

一、对象创建的方法：
1、字面量 – let o = {}
2、new关键字 – let o = new Object()
3、Object.create(optios)

二、对象的属性
1、javascript中的对象既有自由属性，也有继承而来的属性。当查找一个属性时，对象本身没有该属性，会去查找该对象的原型对象，如果该原型对象没有此属性，但这个对象也有原型，会继续查找原型，直到原型为null。

2、赋值。当给一个对象的属性赋值时，如果该对象有此属性时，会改变该属性；如果没有此属性时，会添加该属性并赋值；如果继承的原型中有此属性，那么该继承的属性，会被新的同名属性所覆盖。

3、删除。delete obj.proName

4、属性检测

1）in操作符。检测对象本身的属性以及继承的属性

let o = {x: 1}
"x" in o   // true
"y" in o   // false

2) hasOwnProperty()。检测对象本身的属性，不会检测继承的属性。

let o = {x: 1}
o.hasOwnProperty('x')   // true
